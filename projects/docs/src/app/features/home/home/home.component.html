Angular Library Documentation

This library aims at solving the following problem: easily generate a comprehensive documentation for libraries.

  • Get the API documentation directly from the code.
  • Create and show examples: the code that is shown is the actual code that is running => less copy/paste.

This web page has been built with this library!

Get Started

  1. Install npm install --save-dev ng-documentation
  2. Create a documentation application with the Angular CLI
  3. Setup helper tools

    Add the following to the scripts section of your package.json.

    "postinstall": "cp -R node_modules/ng-documentation/scripts .",
    "build:docs": "./scripts/generate-doc.sh PATH_TO_THE_ASSETS_FOLDER PATH_TO_THE_TSCONFIG_JSON_FILE NAME_OF_ANGULAR_PROJECT && ng build NAME_OF_ANGULAR_PROJECT",
    "watch:docs": "./scripts/watch-docs.sh PATH_TO_THE_ASSETS_FOLDER/documentation PATH_TO_THE_TSCONFIG_JSON_FILE NAME_OF_ANGULAR_PROJECT"
  4. Add the styles to your Angular project styles property in the angular.json. styles: [ "node_modules/ng-documentation/ng-documentation.css" ]
  5. When you want to live serve the documentation app, you must generate the documentation files: npm run watch:docs
  6. When you want to build the documentation app, you must generate the documentation files: npm run build:docs
  7. Configure the documentation
